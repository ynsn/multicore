concept  mtc::allocator
 The mtc::allocator concept is satisfied if Allocator is a type that satisfies all of the requirements of a C++ Allocator.
 
concept  mtc::allocator_for
 The mtc::allocator_for concept is satisfied if Allocator satisfies the mtc::allocator concept and allows allocation and deallocation of objects of type T.
